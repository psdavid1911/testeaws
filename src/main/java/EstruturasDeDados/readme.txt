As classes aqui criadas levam em consideração implementações de interfaces em 
java, isto é, você pode utilizar-se de cast para fazer as conversões adequadas,
darei alguns exemplos comuns

   (ArrayList<T>) de uma lista 'List<T>'
   (HashSet<T>) de um conjunto 'Set<T>'
   (...)

A documentação das interfaces, abstrações e algoritmos - a maior parte deles - 
se encontra na página oficial do java.
